# 📊 Hybrid Workforce Attendance Analysis

## 📌 Project Overview

This project analyzes employee attendance data to understand hybrid work behavior, monitor workforce trends, and support data-driven decision-making.

The HR team stored attendance records across three separate monthly files (April–June 2022), making it difficult to generate consistent insights and manage workforce planning effectively.

To solve this problem, I built an interactive Power BI dashboard that centralizes attendance data and provides actionable insights for office space optimization and hybrid work policy decisions.

---

## 🎯 Business Objective

This project aims to answer the following business questions:

- What is the overall employee attendance rate?
- How frequently do employees work from home (WFH)?
- Which days have the highest in-office attendance?
- Are there any noticeable sick leave trends?
- How can office space be optimized for a hybrid workforce?

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| 📊 Power BI | Dashboard development & data visualization |
| 🔄 Power Query | Data cleaning & transformation |
| 📈 DAX | KPI calculations |
| 📑 Excel | Source data |

---

## 🔧 Data Preparation

### Data Cleaning & Transformation

- Combined three monthly attendance files into one dataset using Power Query.
- Standardized data formats and corrected data types.
- Removed duplicate and missing records.
- Created a centralized dataset for workforce analysis.

---

## 📊 Key Performance Indicators (KPIs)

- ✅ Attendance Rate (%)
- 🏠 Work From Home Rate (WFH %)
- 🩺 Sick Leave Rate (SL %)

---

## 📈 Key Insights

### Attendance Performance

- Average attendance rate: **91.76%**
- Attendance dropped to **77.92%** in mid-May.

### Hybrid Work Patterns

- Average WFH rate: **9.93%**
- Friday had the highest WFH rate (**12.96%**).
- Thursday had the second-highest WFH rate (**11.34%**).
- Monday and Tuesday had the highest in-office attendance.

### Sick Leave Trends

- Average sick leave rate: **1.10%**
- Sick leave increased in late May and early June.

---

## 💡 Business Recommendations

### 🏢 Optimize Office Space

Implement a voluntary Friday WFH policy to reduce office occupancy and improve space utilization.

### 🤝 Schedule Team Activities Strategically

Organize meetings, workshops, and collaborative activities on Mondays and Tuesdays when attendance is highest.

### 🩺 Prepare for Seasonal Sick Leave Peaks

Develop contingency plans and cross-train employees to minimize productivity disruptions during periods of increased sick leave.

---

## 🚀 Project Impact

This dashboard transformed fragmented attendance records into a centralized self-service reporting solution.

It enables HR teams to:

- 📊 Monitor workforce trends efficiently
- 🏢 Improve office space planning
- 📅 Support hybrid work policy decisions
- 📈 Make data-driven workforce planning decisions

The project also establishes a repeatable framework for future workforce analytics projects.
